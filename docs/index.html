STIX Theme for Hugo

unofficial Scientific and Technical Information Exchange (STIX) theme for Hugo. The theme uses the STIX Fonts for body text (hence the theme's name), Helvetica for headings, DejaVu Sans Mono for code, and KaTeX for math.

The goal is to create a simple theme for Hugo specifically made for technical blogs and homepages.

View the STIX Theme for Hugo on GitHub.
